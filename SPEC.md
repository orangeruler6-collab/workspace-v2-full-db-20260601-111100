## 设计方向

**关键词**：赛博朋克（收敛版）| 专业感 | 极简功能感 | 深色沉浸

不是那种"花哨的霓虹灯"风格，而是沉稳的、有质感的科技感。
像一个专业创作工具，而不是一个玩具。

## 字体

- **正文**：`PingFang SC, Microsoft YaHei, -apple-system, sans-serif`
- **数字/时间**：`SF Mono, Consolas`（等宽，用于时间戳、数据）
- **字号层级**：标题 20px / 正文 14px / 辅助 12px / 标签 10px

## 布局

```
┌─────────────────────────────────────────────┐
│  HEADER: Logo + 状态 + 时间                 │
├─────────────────────────────────────────────┤
│  TABS:  [小工具] [AI生图] [运营] [排期] ... │
├─────────────────────────────────────────────┤
│                                             │
│  CONTENT: 卡片网格 或 全宽组件               │
│  - 卡片网格：auto-fill, minmax(340px, 1fr)  │
│  - 间距：24px  padding, 20px gap           │
│                                             │
└─────────────────────────────────────────────┘
```

**最大内容宽度**：1400px，居中
**Header**：固定，不吸顶
**Tab bar**：背景 surface，圆角 12px，padding 4px

## 卡片规范

```
┌─────────────────────────────────┐
│  [标题]        [副标题]  [状态] │  ← card-hdr
├─────────────────────────────────┤
│                                 │
│  表单内容 / 操作按钮             │  ← card-body
│                                 │
└─────────────────────────────────┘
```

- 背景：`--bg-card`
- 边框：`1px solid --border`，圆角 16px
- 悬停：边框变亮 `--border-bright`，加微弱 glow
- 内边距：card-body 16px 18px
- 状态标签：背景 `--primary-dim`，青色文字

## 功能清单

### Tab 1：小工具
- 转写（视频链接 → 文本）
- 评论助手（账号+主题 → N条评论）
- 文案创作（账号+指令 → 脚本）
- 图生文（图片 → 描述词）
- 向量搜索（账号+场景+关键词 → 素材）

### Tab 2：AI 生图
- 文生图（描述词 → 图片）
- 图生图（图片+描述 → 重绘图）

### Tab 3：运营
- 执锝表单（链接/CSV → 分析）
- 评论导出（Cookie → 评论列表）

### Tab 4：向量库（可视化组件）
### Tab 5：排期（看板组件）
### Tab 6：工作流（画布组件 - 待重建）

## 交互原则

1. **所有操作结果自动复制到剪贴板**，显示 toast 提示
2. **加载状态**：按钮显示加载中文字 + 禁用，不可重复点击
3. **空状态**：占位符提示"请输入xxx"
4. **错误状态**：toast 显示具体错误信息
