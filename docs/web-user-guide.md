# 乌萨奇工作平台 Web 使用教程

本文档面向日常使用者，用来快速了解平台入口、核心流程和各模块的正确用法。平台主线是：发现内容机会 -> 拆解/生成文案 -> 管理素材与排期 -> 后期处理 -> 发布与复盘。

## 1. 登录与基础操作

### 1.1 进入平台

1. 打开 Web 地址：`http://localhost:3000/`。
2. 使用团队账号登录。
3. 登录后默认进入「账号热榜」页面。

### 1.2 页面布局

平台主要由三部分组成：

- 左侧侧边栏：切换不同业务模块。
- 中间主工作区：当前模块的操作页面。
- 右下角乌萨奇 AI：全局 AI 助手，可询问数据、文案、热点、排期，也可以帮你跳转模块。

### 1.3 权限说明

不同账号能看到的模块可能不同：

- 普通成员一般可使用账号热榜、每日热点、文案工作台、排期、AI 生图、后期工具、视频发布、素材库等。
- 组长/部长/管理员可看到更多数据与管理功能，例如运营看板、成员管理、操作日志。
- 如果看不到某个模块，通常是账号权限未开放。

### 1.4 全局提示

平台会通过底部弹窗提示操作结果，例如保存成功、解析失败、生成完成等。

排期交接时，如果别人把任务拖给你，你会收到全局弹窗提示「您收到新的订单」。即使你不在排期页面，也能看到提醒。

## 2. 推荐工作流

### 2.1 从热门内容进入文案工作流

适合想从爆款内容里找选题、拆文案、生成脚本。

1. 进入「账号热榜」。
2. 点击「扫描账号池」拉取监控账号的新内容。
3. 查看榜单里靠前的视频，重点看发布时间、平台数据、AI 提醒和热度判断。
4. 对合适内容点击「进工作流」。
5. 系统会把该内容链接带入「文案工作流」。
6. 在文案工作流里依次运行输入源、转写/解析、汇总、背景搜索、分析、确认创意、生成文案。

### 2.2 从商单 BF 生成文案

适合客户给的是需求文档，而不是视频转写稿。

1. 进入「文案工作流」。
2. 在输入源卡片选择「商单 BF」。
3. 可输入飞书链接，或拖拽 PDF/Word 文件导入。
4. 在补充要求里写清楚品牌、平台、账号、风格、禁忌点。
5. 点击运行输入源。
6. 后续按工作流节点继续生成创意和脚本。

注意：扫描版 PDF 如果没有可提取文字，可能需要 OCR；解析失败时建议换成 Word、可复制文字版 PDF，或先把内容粘贴到补充文本里。

### 2.3 从排期交接到后期

适合团队协作管理任务。

1. 进入「排期看板」。
2. 选择组别。
3. 在「按人」视图查看每个人的任务。
4. 文案写完后，把任务卡拖到后期同事名下。
5. 被交接人会收到全局弹窗提示。
6. 后期完成后点击任务卡上的勾号，标记完成。

## 3. 账号热榜

「账号热榜」用于监控抖音、B 站等账号的新内容，把值得关注的内容排到前面。

### 3.1 维护账号池

1. 打开「账号热榜」。
2. 在右侧「账号池」点击「维护」。
3. 填写账号显示名。
4. 输入账号 ID、主页链接或抖音主页分享口令。
5. 选择平台与分区。
6. 点击「加入账号池」。

建议：账号显示名用团队内部好识别的名称，例如「某某-游戏杂谈」「某某-数码」。

### 3.2 扫描账号池

点击「扫描账号池」后，系统会按账号池抓取新增内容。

当前逻辑：

- 同一作品不会重复入库。
- 重复扫描只更新平台能返回的数据。
- B 站优先看播放量、点赞等数据。
- 抖音如果抓不到播放量，就优先展示点赞、评论等可获得字段。
- 发布时间越近，排序权重越靠前。
- 超出时间窗口的内容会进入历史归档，避免老视频长期霸榜。

### 3.3 查看榜单

榜单默认展示当前时间窗口内的内容，重点关注：

- 标题与账号。
- 发布时间。
- 平台。
- 播放、点赞、评论等指标。
- AI 提醒。
- 热度或异常增长判断。

如果使用 AI 定位或筛选后找不到全部内容，点击「返回全部」恢复完整榜单。

### 3.4 进入文案工作流

榜单卡片上点击「进工作流」，系统会把该作品链接带到文案工作流输入源。你只需要继续运行工作流即可。

## 4. 每日热点

「每日热点」用于查看当日热点、筛选可做选题，并辅助进入后续创作。

常见用法：

1. 进入「每日热点」。
2. 刷新或查看当天热点列表。
3. 选择与账号定位相关的热点。
4. 结合热点摘要、来源、平台热度判断是否值得做。
5. 将合适热点转入创意或文案流程。

建议：热点不要只看热度，要判断和账号人设、受众、平台调性是否匹配。

## 5. 文案工作台

文案工作台是一个分组入口，包含账号风格、项目、对话写作、草稿、文案工具和文案工作流。

### 5.1 采集首页

用于采集账号样本和风格素材。

使用建议：

- 把代表账号风格的视频、标题、转写稿或案例集中录入。
- 样本越稳定，后续风格迁移越可靠。

### 5.2 账号库

用于查看和维护账号风格资料。

可用来确认：

- 账号定位。
- 常用标题方式。
- 语言风格。
- 内容结构。
- 适合/不适合的表达。

### 5.3 项目库

用于管理商单项目、品牌资料、客户要求和关联账号。

建议把以下内容写清楚：

- 品牌/项目名称。
- 投放平台。
- 目标受众。
- 核心卖点。
- 禁止表达。
- 参考案例。

### 5.4 对话写作

适合用账号风格进行单次写作或改稿。

使用方式：

1. 选择账号风格。
2. 输入需求或粘贴草稿。
3. 让 AI 按对应账号风格生成、改写或优化。

### 5.5 草稿

用于查看和管理历史草稿。

建议：

- 没定稿的内容先放草稿。
- 成稿后再进入发布或排期。

## 6. 文案工具

「文案工具」适合处理单个素材或轻量任务。

### 6.1 抖音转写

1. 粘贴抖音分享口令、视频链接或图文链接。
2. 点击运行。
3. 等待转写完成。
4. 可点击「生成文案」继续创作。
5. 可点击「清洗分段」优化转写稿。
6. 需要同步时点击「写入飞书」。

### 6.2 B 站转写

1. 粘贴 B 站链接、b23 短链或 BV 号。
2. 点击运行。
3. 等待转写完成。
4. 可生成文案或清洗分段。

### 6.3 评论素材

用于根据脚本生成评论区素材。

1. 粘贴视频文案或脚本。
2. 可填写视频链接。
3. 选择评论数量。
4. 选择账号风格。
5. 点击「生成评论」。
6. 生成后可复制全部或写入飞书。

### 6.4 向量搜索

用于从文案库、案例库或历史资料里找参考。

1. 输入搜索内容。
2. 选择账号、类型等筛选条件。
3. 点击「搜索文案库」。
4. 查看相似素材和可参考表达。

## 7. 文案工作流

「文案工作流」适合复杂、多步骤的脚本生产，是目前最重要的文案生产入口。

### 7.1 基础视图操作

- 使用 `+` / `-` 放大缩小。
- 点击「适应视图」让整个流程回到合适视野。
- 点击「重置」清空当前流程状态，重新开始。

### 7.2 输入源

输入源支持两种类型：

- 素材输入：适合抖音/B 站视频链接、飞书链接等。
- 商单 BF：适合客户需求文档、品牌 brief、PDF/Word 本地文件。

可输入：

- 抖音链接。
- B 站链接。
- 飞书文档链接。
- PDF / Word 文件。
- 手动补充文本。

### 7.3 商单 BF 文件导入

1. 切换输入源为「商单 BF」。
2. 拖拽 PDF/Word 到文档区域，或点击「选择文件」。
3. 等待解析完成。
4. 检查解析出来的文字是否完整。
5. 如果有遗漏，在手动补充区补充。

注意：

- Word 文档通常比扫描版 PDF 更稳定。
- 扫描版 PDF 可能没有可提取文字。
- 飞书链接需要账号授权可读。

### 7.4 节点运行顺序

建议按以下顺序推进：

1. 输入源：收集链接、文档和补充要求。
2. 转写/解析：提取视频或文档内容。
3. 汇总：整理核心信息。
4. 背景搜索：补充外部背景，但要注意噪音。
5. 平台素材：结合平台资料和历史案例。
6. 分析报告：判断可用信息、风险点、内容方向。
7. 确认创意：选择风格、角度和脚本方向。
8. 生成文案：输出可编辑脚本。
9. 发布推荐：生成标题、简介、封面方向等。

### 7.5 风格选择

创意阶段会涉及账号风格。如果没有选择，系统会使用默认风格。

建议：

- 商单脚本优先选目标投放账号风格。
- 热点改编优先选最接近受众的账号。
- 不确定时先默认生成，再用账号风格做二次改写。

### 7.6 常见问题

- 卡在运行中：刷新页面前先看是否已有输出；如果长期无响应，重新运行当前节点。
- 背景搜索噪音多：以转写原文/BF 为准，背景搜索只做补充，不要把未确认传闻当事实。
- 飞书链接读不了：确认飞书 CLI/授权已登录，且当前账号有文档权限。
- PDF 解析失败：优先换 Word、文字版 PDF，或手动粘贴正文。

## 8. 创意看板

「创意看板」用于沉淀选题、创意方向和待处理想法。

常见用法：

1. 新建创意卡片。
2. 填写标题、来源、内容方向。
3. 补充参考链接或素材。
4. 根据状态推进：待处理、处理中、已完成等。
5. 需要写作时转入文案工作流或文案工具。

建议：创意卡片不要只写标题，要写清楚「为什么值得做」「适合哪个账号」「切入角度是什么」。

## 9. 排期看板

「排期看板」用于团队任务分配、交接和完成确认。

### 9.1 按人视图

按人视图适合看每个人手上有哪些任务。

操作：

- 点击组别切换不同小组。
- 点击「+ 添加」给某人添加任务。
- 拖动任务卡到另一个人名下完成交接。
- 点击勾号标记完成。
- 点击编辑按钮修改任务。
- 点击删除按钮删除任务。
- 按 `Ctrl + Z` 或点击「撤销」撤回上一步操作。

### 9.2 按周视图

按周视图适合看账号和日期分布。

操作：

- 点击左右箭头切换上一周/下一周。
- 点击「今天」回到当前周。
- 拖动任务调整日期或账号。

### 9.3 AI 导入排期

适合一次性导入多条任务。

1. 点击「AI 导入」。
2. 粘贴任务描述，例如「某项目素材代做 24 条」。
3. 点击「解析」。
4. 选择拆分方式，例如每人 4 条、每人 6 条或保持单条。
5. 确认预览无误后点击「确认导入」。

### 9.4 交接提醒

当任务从 A 拖给 B 后：

- A 看到本地移动成功提示。
- B 会收到全局弹窗「您收到新的订单」。
- 通知弹出后会自动标记已读。

## 10. 运营看板

「运营看板」用于查看流水、毛利和组内经营数据。

### 10.1 查看数据

1. 选择组别。
2. 查看总流水、毛利、完成率等概览。
3. 查看账号流水列表。
4. 可刷新数据或清空当前数据。

### 10.2 录入流水

支持自然语言录入：

1. 展开「录入流水 / 导入 Excel」。
2. 粘贴确认信息，例如项目名、账号、金额、月份。
3. 点击「规则解析」或「智能解析」。
4. 检查解析预览。
5. 点击「写入数据库」。

### 10.3 AI 数据分析

点击「AI 分析」，可以询问：

- 这个月总毛利是多少？
- 哪个平台流水最高？
- 哪些账号贡献最大？
- 和上月相比有什么变化？

## 11. AI 生图

「AI 生图」用于生成封面图、参考图、角色图或视觉素材。

### 11.1 文生图

1. 选择「文生图」。
2. 输入画面描述。
3. 选择模型。
4. 选择比例、数量等参数。
5. 点击「开始生成」。
6. 生成后可预览、下载或复制链接。

### 11.2 图生图

1. 选择「图生图」。
2. 上传或拖拽参考图。
3. 输入想要变化的方向。
4. 点击生成。

### 11.3 历史记录

进入「历史」可以查看过往生成结果，并可复用参数、下载图片或删除记录。

## 12. 后期工具

「后期工具」用于处理剪辑、格式转换、音视频辅助等后期相关任务。

使用建议：

- 先确认素材是否已经进入素材库。
- 再按后期需求选择对应工具。
- 处理完成后可回到素材库或视频发布模块继续使用。

## 13. 视频发布

「视频发布」用于整理发布信息，并生成平台发布任务。

### 13.1 上传视频

1. 点击或拖拽视频到上传区域。
2. 支持本地视频，也可从素材库导入。
3. 可继续添加多个视频。

### 13.2 填写发布信息

需要填写：

- 标题。
- 简介。
- 标签。
- 封面信息。
- 发布时间：立即发布、定时发布或保存草稿。

可点击「AI 推荐」让系统根据视频和账号风格生成发布信息。

### 13.3 选择发布账号与平台

1. 在右侧搜索或选择发布账号。
2. 选择平台。
3. 可点击「已登录」或「全选」快速选择平台。
4. 点击「生成发布任务」。

当前说明：后端接入 OpenCLI 后，会进一步打开对应平台发布流程。

## 14. 素材库

「素材库」用于统一管理视频、图片、音频、文档等素材。

### 14.1 上传与管理

常见操作：

- 上传素材。
- 新建文件夹。
- 刷新列表。
- 搜索素材。
- 按类型、标签、文件夹筛选。
- 预览、编辑、下载、删除素材。

### 14.2 编辑素材信息

可编辑：

- 素材名称。
- 分类。
- 标签。
- BGM 标签等。

### 14.3 AI 标签

点击「AI 标签」可让系统识别素材内容并生成标签，方便后续搜索。

### 14.4 剪辑篮

可把多个素材加入剪辑篮，方便后续集中交给后期或发布使用。

## 15. 智能采片

「智能采片」用于采集外部素材，并沉淀到素材库。

建议流程：

1. 输入或导入素材来源。
2. 执行采集。
3. 检查采集结果。
4. 选择有价值的素材入库。
5. 后续在素材库中打标签和分类。

## 16. 向量库

「向量库」用于检索历史资料、文案案例、商单资料和知识库内容。

适合场景：

- 找历史相似脚本。
- 找某账号过去的表达方式。
- 找项目资料。
- 找素材标签和案例。
- 辅助文案工作流做背景参考。

使用建议：搜索词尽量具体，例如「某游戏 商单 反转开头」「某账号 数码测评标题」。

## 17. 成员管理与操作日志

这两个模块通常只有管理员、组长或部长可见。

### 17.1 成员管理

用于管理成员账号、角色、权限和在职状态。

常见操作：

- 新增成员。
- 修改显示名、组别、职位。
- 调整权限。
- 启用或停用账号。

### 17.2 操作日志

用于查看平台内关键操作记录，方便追踪谁在什么时候做了什么。

## 18. 右下角乌萨奇 AI

右下角 AI 是全局助手。

你可以问：

- 帮我打开文案工作流。
- 这条链接带进工作流。
- 今天账号热榜有什么值得看？
- 帮我看排期。
- 帮我分析运营数据。
- 这个 BF 应该怎么拆？

使用技巧：

- 问题越具体，结果越好。
- 如果要让它跳转模块，直接说模块名。
- 如果要处理链接，最好把链接一起发给它。
- 它会结合当前模块回答，但复杂任务仍建议进入对应模块操作。

## 19. 常见问题

### 19.1 为什么我看不到某个模块？

一般是权限问题。请联系管理员检查账号权限。

### 19.2 为什么飞书链接读取失败？

常见原因：

- 当前账号没有文档权限。
- 飞书授权过期。
- 链接不是文档正文链接。
- 本地飞书 CLI 未登录。

处理方式：重新授权飞书账号，确认文档可访问，再重试。

### 19.3 为什么 PDF 解析失败？

常见原因：

- PDF 是扫描版，没有可复制文字。
- 文件加密。
- 文档图片过大或格式异常。

处理方式：优先使用 Word 或可复制文字版 PDF；必要时先 OCR，再导入。

### 19.4 为什么账号热榜有些字段不显示？

不同平台可抓取的数据不同。系统只展示能稳定抓到的字段，抓不到的字段不会硬显示，避免看起来像页面 bug。

### 19.5 为什么榜单里老视频不一直排第一？

热榜有时间窗口。发布时间越近权重越高，超过窗口的内容会归档，避免十几天前的视频长期占据当前榜单。

### 19.6 生成文案卡住怎么办？

建议按顺序排查：

1. 看当前节点是否已有输出。
2. 只重新运行当前失败节点，不要直接重置全流程。
3. 检查输入源是否为空。
4. 检查 BF/链接是否可读。
5. 如果多次失败，再刷新页面重试。

## 20. 最佳实践

- 热榜只负责发现机会，不要把热度当成唯一判断。
- 文案工作流里，转写原文和 BF 优先级高于背景搜索。
- 背景搜索结果必须和原始输入互相验证，不能直接把未证实信息写进脚本。
- 商单内容要写清楚品牌禁忌、卖点和平台要求。
- 排期任务交接尽量用拖拽，让系统自动通知对方。
- 后期完成后及时打勾，避免任务长期挂起。
- 素材入库时尽量补标签，否则后续很难搜。
- 生成图、发布文案、评论素材都建议人工复核后再对外使用。
